# 手机版
> Q：存档导出然后导入的时候显示完整性检验失败  
A：大部分原因是没粘贴完全。QQ和微信有长度限制，推荐使用发TXT，云存档，云文档等等传递存档。（有道云笔记本，腾讯文档等等）<br>注意某些手机自带的输入法剪贴板可能也有长度上限。


- 猫国建设者移动版于2016年底开始开发。游戏在2017年初进入了公开测试。目前IOS和ANDROID两个版本都已经加入了中文，在选项里面切换即可。

- android版本（**比网页滞后一个版本**）￥12：<a href="https://play.google.com/store/apps/details?id=com.nuclearunicorn.kittensgame" target="_Blank">https://play.google.com/store/apps/details?id=com.nuclearunicorn.kittensgame</a> （需魔法，猫国群内有分享）

- iOS版本（**比网页滞后好几个版本**） ￥18：<a href="https://apps.apple.com/cn/app/kittens-game/id1198099725" target="_Blank">https://apps.apple.com/cn/app/kittens-game/id1198099725</a>（**国区已经下架了，需要外区下载了**）
 - iOS账号地区改成非中国地区 搜索kittens game

<ul>
<li>离线进程</li><li>通知</li><li>进度条和预制作</li><li>完全兼容现有的存档</li>
</ul>

注意事项：
- IOS版本的更新速度通常比较慢
