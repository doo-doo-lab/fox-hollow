# 污染
- 游戏1.4.8.5才加入的污染系统。
- 科学研究完<a href="?file=001-猫咪百科/03-科学/01-科学#生态学">生态学</a>后，且二氧化碳浓度 > 1ppm，游戏上方会出现污染工具栏。
 - $二氧化碳\left (ppm  \right ) =\frac{污染量}{100000} $
 - 污染级别增加 污染的工具栏会变为红色提示。
 - 没研究完生态学，也会有累积二氧化碳，到达50PPM可以在猫薄荷产量看到污染减少的百分比产量。
- **简单来讲，用污染换发展快有和环保发展极慢只能二选一**。
 - 比如要环保，研究完<a href="?file=001-猫咪百科/04-工坊/01-升级#碳封存">碳封存</a>后，开工厂的**自动化**，加上关磁电机（因为磁电机的污染排放最大）<br>这样又缺电，又少了磁电机的全局生产加成。关不彻底不能彻底解决污染，所以发展和污染只能二选一。

##  污染改善:
- 唯一能**减轻污染的建筑**只有升级<a href="?file=001-猫咪百科/04-工坊/01-升级#碳封存">碳封存</a> 后。
- **污染改善**（每秒） =  -10 \* 每个工厂自动化的数量

---
##  清洁能源系数:
- **清洁能源系数**：$ \frac{(太阳能 + 水电站 +反应堆 + 卫星)总瓦数 }{(太阳能 + 水电站 + 反应堆 + 卫星 + 磁电机 + 蒸汽工房)总瓦数} $
 - 即清洁能源系数最大值为1。
- 注意**卫星**产生能源需要在工坊升级 <a href="?file=001-猫咪百科/04-工坊/01-升级#太阳能卫星">太阳能卫星</a>。

---
##  环境自净:
- **环境自净**（每秒）：二氧化碳浓度 \* 0.05  
- 指污染越严重环境自净(污染消散)的也越快，逐步进入稳态。

---
##  污染排放率
- 即字面意思：排放源的加成。

## 实际污染排放:
- **实际污染排放**（每秒） = 总污染源排放 \*（1 - 0.5 \* 清洁能源系数）\*（1 + 污染排放率） - 污染改善 - 环境自净
- 即二氧化碳浓度最终会因为环境自净的增加而稳定下来。
- 负数会显示环境正在改善。（可能需要关磁电机，得不偿失多点农民就好了）

---
##  污染度级别
- 污染度级别 1 需要 100ppm 的二氧化碳浓度，每提高一级需要10倍的二氧化碳浓度。
- 每个污染度级别，都有对应游戏上方的污染描述。
- 暂时有4个 污染影响：猫薄荷产量减少，幸福度环境影响，减慢小猫出生速度，太阳革命对于 猫薄荷 和 木头 的产量减少。
 - 显示的时候都是四舍五入，游戏计算的话是计算小数的。

###  猫薄荷产量减少
>x = 二氧化碳浓度(ppm)  
y = 猫薄荷产量减少(%)  
二氧化碳 \* 100000 = 污染排放总量  
$\ln_{}{100000} \approx   11.512925465$

$\begin{array}{l} y
  \left\\\{\begin{matrix}
0,x\lt 50\\\\
\frac{2x-100}{5},50 \le  x\lt   100  \\\\
\frac{3900+x}{200}, 100 \le  x\lt 1000\\\\
{\ln_{}{x}+5+ 11.513},x \ge 1000\\\\
\end{matrix}\right.
\end{array}$

###  幸福度环境影响
>x = 二氧化碳浓度(ppm)  
y = 幸福度减少(%)  

$\begin{array}{l} y
\left\\\{\begin{matrix}
0,x\lt 500\\\\
\frac{4x-2000}{125}, 500 \le  x\lt 1000\\\\
\left ( {\ln\_{}{x}+ 11.513} \right )  \times 1.08 ,1000 \le  x\lt 10K\\\\
\left ({\ln\_{}{x}+  11.513} \right ) \times 1.18 ,10K \le  x\lt 100K\\\\
\left ({\ln\_{}{x}+ 11.513} \right )  \times 1.2 ,x \ge 100K\\\\
\end{matrix}\right.
\end{array} $

---
###  减慢小猫出生速度
>x = 二氧化碳浓度(ppm)  
y = 小猫出生速度减少的速率（%）  
出生速度$ =\frac{\text{基础出生速度} }{ 1+y} $

$\begin{array}{l} y
  \left\\\{\begin{matrix}
0,x\lt 5000\\\\
16+0.0168x,5000\le  x\lt 10K  \\\\
\left ( \log\_{10}{x}+5 \right ) \times 111 ,10K \le  x\lt 100K\\\\
\left ( \log\_{10}{x}+5 \right ) \times 120 ,x \ge  100K\\\\
\end{matrix}\right.
\end{array}$

###  太阳革命对于 猫薄荷 和 木头 的产量减少
>x = 二氧化碳浓度(ppm)  
y = 太阳革命对于 猫薄荷 和 木头 的产量减少(%)

$\begin{array}{l} y
  \left\\\{\begin{matrix}
0,x\lt 100K\\\\
\frac{x-100K}{9K},x \ge  100K\\\\
\end{matrix}\right.
\end{array}$
