## 写在前面
- 建议、BUG等问题可以在猫国Q群询问或者私Cheney。
- 可以通过勾选/取消勾选最上方的**启用小猫珂学家**的方式，来控制整个脚本是否运行。
  - 对于每个分类和每个小项目，也都可以用同样的方式来控制。（尽量用啥就勾啥）
- 珂学家**每2秒**运行一次。（跟随游戏天）
  - 如果有时间通量并勾选时间操纵的光阴似箭，游戏会变成每2秒/天 → 每1.33秒/天<br>珂学家也会变成每隔1.33秒运行一次
- 当你第一次使用小猫珂学家的时候，阅读使用说明，然后再根据需求逐个开启珂学家的功能。
  - **最简单就是全勾所有大项目就行了**（**不需要你怎么研究~**）
    - 你只需要勾需要的大项目，项目内max和启用已经帮你勾选好了
  - 默认设置可以一直以最快速度用到毕业
    - 政策的话可以自己勾选自动政策
    - 已经勾选的项目都已经是自动化了
    - 并没有给**萌新**开启太多的优化，保留前面的可玩性，会随着周目后越来越智能。
  - 参考默认设置无改动的冲出轨道时间，黄金比例 80 年，神圣比例 50 年，维特鲁威猫 25~30 年，文艺复兴 7 年
- **默认设置基本算是挂机的推荐设置**，**如需修改请参考说明书**。
  - 比如工艺的项目按照初始的推荐**勾上**，挂机效率会比较高。
- **想知道小猫干了什么**，可以点<font color=red>小猫总结</font>。
- 后面比如**文化、煤、科学、黄金、钛**会一直在**上限花不出去**，**属于正常现象**。（木桶效应要等产量低的资源）
  - 因为要保留低产量的资源，高产量资源只能爆仓。
- 日志会比较吃性能，电脑性能特别不好的，推荐根据需要过滤不用的。

## 触发条件
- **可以把鼠标移动在该项目会上显示触发条件对应的资源**。
- 触发条件：表示该项目的触发条件资源 / 触发条件资源上限 达到多少时，会触发小猫珂学家。
 - 例如，猎人狩猎的默认触发条件是0.98，那么你的**当前喵力 / 喵力上限  >= 0.98**，才会**触发珂学家**猎人狩猎。
 - 例如，你设置了营火触发条件是0.6，比如猫薄荷田那就是需要 你当前**猫薄荷数量** / **猫薄荷上限** >= **0.6** 才会**触发珂学家**建筑猫薄荷田。
 - 例如，你设置了营火触发条件是**0** 或者**仓库、贸易船**这种触发条件**无**对应的资源的话，是**资源够就会建造**。

### 营火

> 这个分类支持触发条件的设置，默认的触发条件是0。
>> 注意小猫珂学家只有寒冷冬季（- 90%）猫薄荷产量 > 0，才会使用**猫薄荷**时造猫薄荷田等等。

- 只要勾选某个地面建筑物，珂学家就会在满足触发条件时自动建造。
  - 鼠标移动到该项目会显示对应的触发资源
  - 无触发资源的表示资源足够就会建造
- 每个建筑物的右侧可以设置最大值，珂学家不会建造超过这个最大值的数量。<br>
 - 默认情况下最大值是-1，这意味着没有最大值。

### 工艺

> 这个分类支持触发条件的设置，默认的触发条件是0.95，所有资源的默认消耗率为0.6。  
>> - 注意小猫珂学家用**猫薄荷制作木材**时，会根据你的**寒冷冬季**（**- 90%**）猫薄荷产量。
 - 如果没超过触发条件且勾选AI平衡时，寒冬猫薄荷产量 ＞ 0 时才会制作。
 - 如果超过达触发条件，小猫珂学家会根据你设置：消耗率和库存，会再多帮你保留1个季度寒冬需要的猫薄荷数量。
 - 你可以根据统计右边**概览**看到你的**寒冷冬季猫薄荷产量**。

- 工艺分成2种模式：（互不冲突）
 - **优先**判定**触发条件**，不到达触发条件**再**判定**AI平衡**的公式。
 - 推荐能勾AI平衡尽量勾AI平衡，AI平衡的效率比触发条件高点。<br><br>
- **触发条件** 
 - 当前资源数量 / 资源上限 ＞ 触发条件 时，珂学家就会根据原材料的制作率来制作下一级资源。
   - 鼠标移动到该项目会显示对应的触发资源
 - 默认的触发条件是0.95，默认制作率为0.6，即制作60%的资源，保留40%。
   - 比如你猫薄荷上限15K，15K \* 0.95 = 14.25K 时才会触发珂学家精炼猫薄荷，精炼 14.25K \* 0.6 = 8.55K的猫薄荷。
   - **没有对应资源上限的且没勾选AI平衡**，比如脚手架，原材料木梁是无库存上限的<br>即木梁够50⇒脚手架就会直接木梁⇒脚手架合成。
- **AI平衡**（**简单理解成实时自动的自动AI模式**，推荐默认设置即大部分勾选AI平衡，触发条件的效率比较低）
 - 勾选AI平衡的资源。（如果有对应资源上限会先判断触发条件，再判断AI平衡）
   - 固定为消耗资源产量值的50%，保留50%资源的产量值。
     - 只要原材料或者勾选的资源数目**变动**都有可能触发AI平衡。详情公式：<a href="?file=004-第三方工具/02-小猫科学家#工艺AI平衡公式" target="_Blank">工艺AI平衡公式</a>
   - AI平衡就是根据你的**制作工艺效率、原材料的数量、成品拥有的数量趋于平衡**。
     - 比如矿物 → 石板，石板变相少的话就珂学家就会合成产量值的50%，石板变相多的话珂学家就不会合成。
- **注意：当 钢 或者 金属板 的AI平衡勾选时会有另外一种逻辑来确保金属板或钢中的1个变得特别多。**

钢比较特殊，它的触发条件对应资源是煤，跟铁无关。即只要煤不到触发条件，铁在多都不会触发。

<br>可以支持比较复杂的设置：（**推荐默认即可**，调教较为复杂）<br>**默认设置下勾AI平衡挂机效率会比单纯触发条件的高很多**
- 通过触发条件旁边的**资源**按钮，可以配置每一种资源的**库存**，以及**消耗率**。
- 消耗率
 - 默认情况下，所有资源的**消耗率**都是0.6，都只会使用60%的资源，保留40%。（**注意仅作用在工艺的触发条件**）
  - 设置  1 为 全部消耗，不会保留。
  - 设置 0.5 为 制作50%，保留50% 
  - 设置  0 为 不消耗。
- 库存（给你手动的设置，挂机推荐不设置库存）
 - 毛皮有着默认的**库存**，数值是500，只有毛皮数量超过500**且触发条件或者AI平衡触发了**后总会给你留下1000的毛皮。
 - 其他的资源默认库存都是0，如果有需要，可以手动分别设置。
 - **即使消耗率设置成1**，小猫珂学家在制作时也会保留你设置好的库存数量。
 - **注意这里设置的库存会影响整个珂学家**：
   - 比如工坊升级或者营火建筑等等，比如钛设置1000，工坊某个升级需要 500 钛，即 1500 钛的时候才会升级解锁。
 - 默认 羊皮纸 不勾AI平衡，触发资源为无、**即不勾AI平衡**时资源够了就会合成、资源 毛皮 的库存 500，
  - 意思是只要 毛皮＞500 就会把大于 500 的毛皮的数量全部用去制作羊皮纸。

**如果某些资源到达上限时，资源对应的工艺制作会不工作**。（比如木材满了，猫薄荷不会被珂学家精炼成木材）

#### 工艺限制公式
<details>
<summary><mark><font color=darkred>点击查看工艺限制公式：</font></mark></summary>
限制公式:  
$\gamma $= 工艺制作效率  
$\alpha$ = 珂学家将消耗的原材料数量  
$\alpha =\frac{\text{现有原材料数量} - \frac{\text{目标材料数量} \times \text{制作需求}}{1+\gamma } }{2} $  
制作次数 = $\frac{\alpha}{\text{制作需求}} $  
最后制作目标材料数量 = $\frac{\alpha\times 制作需求 }{1+\gamma} $
</details>

---

公式简单理解:  
当原材料数量 > 目标资源通过等效成原材料数量，珂学家就会使他们平衡。（前提能制作1次以上）  
比如你的工艺制作效率 +300%，木材有 5601个，木梁有120个，175 木材转化一次木梁：  
120 \* 175 = 21000 ⇒  21000 / (1 + 300%) = 5250 // 把木梁转化成木材数量<br>(5601-5250) / (2 \* 175) ⇒  351/350 = 1 (去尾法) // 多出来的材料制作50%保留50%  
即代替你点击1下制作木梁，制作4个木梁。

>> 如果目标材料有多个原材料取最低值。  
比如钢的原材料为煤和铁，制作需求都是100，最后制作的次数讲将煤和铁分别代入取最低值。

### 升级

> 这个分类下整合了各种一次性的升级项目。

- **升级**：
 - 在资源足够时自动进行工坊升级。
   - （勾限制后，不需要的升级小猫珂学家会无视）：
   - 比如 钢铁工厂、小猫工程师相关的升级、工房自动化的相关升级。
- **科技**：
 - 在资源足够时自动研究科技。
- **政策**：
 - 点击读取会获得你现在抉择的政策列表，下周目会自动抉择在列表里的政策。
- **探险队出发！**：
 - 在条件满足、喵力足够时自动派出探险者寻找可供贸易的种族。
- **探索星球**：
 - 在资源足够时自动探索新的星球。（可以设置触发条件，就是只会到第几个星球）
- **建筑**：
 - 在条件允许时自动升级建筑。
<br>这里的升级指的是用新的建筑完全替代原来的建筑。
<br>例如，建造数据中心以取代图书馆，而不是建造更多数量的图书馆。

### 贸易

> 这个分类支持触发条件的设置，默认的触发条件是0.98。<br>

只要勾选某个种族：
- 珂学家就会在**黄金** / **黄金上限** ≥ **触发条件**，且购买的 资源 / 资源的上限 ≥ 触发条件时才会自动和它进行贸易。<br>比如狮鹫：木材且黄金、斑马：黄金、利维坦：黄金且难得素
 - **季节**设置：
   - 对于大部分种族而言，珂学家默认已经设置好，只会在最佳的季节和它贸易。
<br>比如不勾的季节，无论触发条件如何小猫珂学家是不会进行交易的。
   - 点击每一个种族旁边的**季节**，可以手动分别设置。<br>
 - **限制**设置：（类似工艺限制，推荐挂机勾上，当作AI模式）
   - 勾选限制后，每当珂学家工作会分别计算贸易付出的资源、贸易获得的资源的时间，如果**贸易付出的资源比贸易获得资源更容易获得，那么贸易才会进行**。
     - 类似工艺制作的勾选限制一样，勾选限制后会无视触发条件。<br>换算后每秒贸易获得的资源比产出的资源多，小猫珂学家才会触发限制，即贸易一部分。
<br>比如开启斑马的限制后贸易获得的 铁  ＞ 煅烧炉 + 熔炉 铁产量 小猫珂学家才会进行贸易。 

- 注意：
 - 如果出售有上限的资源(除了钛和悲伤)到达上限时，资源对应的贸易种族会不工作。
 - **因为贸易导致对应资源上限时，会导致工艺或者其他功能失效**。（刷新或者重新勾选贸易功能）

### 宗教

> 这个分类支持触发条件的设置，默认的触发条件是0。<br>

只要勾选某个宗教选项，珂学家就会在资源足够时自动学习。<br>
另外，宗教分类有一些**附加**功能：<br>
- 如果开启**优先最佳独角兽建筑**
 - 珂学家就只会建造目前提高独角兽产率性价比最高的建筑。
- 如果开启**赞美太阳**，
 - 在单独设置的触发条件满足时，自动赞美太阳。默认的触发条件是0.98。
   - 当处于默认条件0.98时，太阳革命的加成低于根据顿悟推算的太阳加成会把触发条件当成0。（加速获得虔诚）
- 如果开启**赞美群星**，需同时满足以下2个条件：
 - 仅在 当前信仰 / 信仰上限 ＞ 0.98
 - 满足设置的触发条件。触发条件简单理解：
   - 如果赞美群星后，存满信仰点击赞美太阳，得到太阳革命的加成 / 1000%（太阳革命上限）＞ 触发条件 才会触发小猫珂学家的赞美群星。
 - 次元超越等级低于10，以下会根据上次赞美群星的虔诚数量
 - 触发条件为0.001，会根据次元超越等级来算触发条件。
- 如果开启**自动超越**，需同时满足以下2个条件：
 - 1.同时满足赞美群星的条件
   - 1.1当前信仰 / 信仰上限 ＞ 0.98
   - 1.2赞美群星完后的赞美太阳后太阳革命效果能恢复到赞美群星触发条件。
 - 2.到达推荐值顿悟。
 - 3.次元超越低于10,会强制带上赞美群星,故会参考赞美群星后猫薄荷产量的

### 时间操纵

> 注意：这个分类下的设置需要在非常深入游戏时才会用到，如果还不太了解这些设置的含义，建议不要开启它们。<br>

**时间加速**：
 - 当时间通量达到触发条件时，启动时间加速，否则关闭时间加速。<br>

**时间跳转**：
- 自动燃烧时间水晶。该功能支持四个分支选项的设置：
 - **触发条件**
   - 表示至少有多少时间水晶时，科学家才会尝试时间跳转；
 - **上限**
   - 表示每次时间跳转最多跳转的年数；
 - **季节**
   - 珂学家只会在勾选的季节内燃烧时间水晶；（注意：当前季节 对应的季节设置没勾，不会触发珂学家燃烧时间水晶）
<br>比如到了夏季，夏季设置没勾，就不会触发珂学家燃烧时间水晶。
 - **周期**
   - 科学家同样只会在勾选的周期内燃烧时间水晶。（注意：当前周期 对应的周期设置没勾，不会触发珂学家燃烧时间水晶）
<br>比如到了卡戎周期，卡戎周期设置没勾，就不会触发珂学家燃烧时间水晶。
<br>比如只停留在红月的设置：红月不勾，其他周期全勾。
<br>比如一直燃烧时间水晶的设置：周期全勾。
 - 另外，珂学家永远不会在时计炉过热时燃烧水晶。
 - 勾上红月会根据热量停红月散热。

注意：小猫珂学家没有后台代码，浏览器后台的时候烧珂学家可能会从每2秒运行一次变成10秒运行一次。（可以打开选项的启用web worker后重新勾选珂学家）

### 猫力资源
> 这个分类下是分配猫咪的设置，可以支持稍微复杂的设置：  
注意**当温暖冬季猫薄荷产量小于0会强制分配农民**。（无视MAX和启用）  
注意**职业的Max为正数才能正确分配**。

- ** 限制 **
 - **不勾选限量**，即当前工作猫咪分配会无视 Max 的数量。
 - **勾选限量**，即当前工作猫咪分配会最多为 Max 的数量。
- ** Max **
 - 1.设置**勾选限量**的最大数量。
 - 2.分配猫咪会根据你打勾的工作来分配，
   - 具体：某个工作的猫咪数量 / 某个工作的Max的数量 哪个工作值得出的最小珂学家就会优先分配哪个
   - 当 Max 数量全部分配完毕后，就会根据**没勾选限量**的工作比值来分配工作。<br>比如猎人Max：2，牧师 Max：3，比如会分配成 猎人：牧师 20：30（2:3）

### 选项

> 这个分类下有一些实用的杂项功能：

- **观测天文事件**：
 - 自动观测罕见的天文现象。用过都说好。
- **举办节日**：
 - 当资源足够时自动举办节日。<br>**举办节日需要羊皮纸获取速度 ＞ 62.5 每秒才会自动举办节日。（珂学家会通过工艺制作效率换算）**
   - 有了玄学嘉年华后最多只会加钟到1年多的持续时间。（即持续时间不会超过2年）
- **强制贸易船船**：
 - 当贸易船数量低于触发条件数量时，珂学家会无视工艺制作分类的设置，即默认贸易船的限制为不勾选。
- **献祭上古神**：
 - 自动给利维坦喂食死灵兽，以延长它停留的时间。
- **狩猎**：
 - 当喵力达到上限的一定比例时，自动派出猎人。可以单独设置喵力的触发条件，默认值是0.98。
- **提拔领袖**：
 - 当黄金和经验值足够时，自动提拔领袖。
- **黑币交易**：
 - 自动在价格 <950 买入黑币，在 > 1050卖出。
   - 触发条件为仅当你的遗物 ≥ 你设置的触发条件时贸易。
- **修复冷冻仓**：
 - 在资源足够时自动修复可修复的冷冻仓。
- **建造大使馆（Beta）**：
 - 当文化到达触发条件时，自动建造大使馆。
- **占满屏幕**：
 - 开启科学家时自动进行UI调整，建议开启。
- **启用蒸汽工房**：
 - 开启时，如果蒸汽工房、磁电机、反应堆因原材料不够关闭时，珂学家会再开启它们。
- **珂学家设置存档**：
 - 提供导出、导入、初始配置的按钮。
- **启用web worker**：
 - 当珂学家处于后台时会全速运行。
   - 比如浏览器后台时，珂学家不能把所有的观测事件全点击。（会漏）
