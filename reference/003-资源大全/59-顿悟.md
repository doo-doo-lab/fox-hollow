# 顿悟
- 顿悟是隐藏资源。(如果想要查看拥有多少,可以在左上角点击齿轮⚙,翻到下面勾选显示隐藏资源)

## 获取
- <strong>顿悟</strong>是由点击<a href="?file=001-猫咪百科/06-宗教/002-太阳教团#赞美群星">赞美群星按钮</a>通过消耗全部虔诚转化成顿悟。
 - 获得数量为：$\frac{\text{虔诚}\times\text{顿悟加成}}{1000000} $
   - 顿悟加成 = $ \left ( 1+次元超越等级 \right ) ^ {2} $  
   - **注意没解锁<a href="?file=001-猫咪百科/06-宗教/002-太阳教团#超越" >超越</a>，也就没有[ x 顿悟加成]，即使有<a href="?file=001-猫咪百科/06-宗教/002-太阳教团#次元超越" >次元超越等级</a>，顿悟加成也为1**。
   - 手动点击赞美群星将获得额外1%的顿悟。

### 用途
- 赞美太阳加成与顿悟有关，并提升<a href="?file=001-猫咪百科/06-宗教/002-太阳教团#赞美太阳">赞美太阳！</a> 的效率。
 - 赞美太阳后面的[ ]里的数值就是赞美太阳的加成。
 - 注：赞美太阳的加成公式：$5\cdot \sqrt{1+80\times顿悟 } -5$
- 也可以消耗一定数量的<strong>顿悟</strong>来提升次元超越的等级。

**注：提早次元超越会损失很多效率。**
