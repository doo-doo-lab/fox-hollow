# 星图
- 通过研究<a href="?file=001-猫咪百科/03-科学/01-科学#天文学">天文学</a>后，<a href="?file=002-常用资料/006-事件介绍">观察天文事件</a>后获得星图，平均每天获得星图数量期望值为天文事件出现的概率。

### 获取
> 注意 <a href="?file=001-猫咪百科/11-挑战#黑暗天空">黑暗天空</a>挑战下只有**<a href="?file=001-猫咪百科/07-空间/03-喵星#卫星">卫星</a>**能获得星图。

- 拥有**任意数量图书馆**或者**数据中心**后，每天有 0.25% 的概率出现<a href="?file=002-常用资料/006-事件介绍">天文事件</a>。
 - 即一开始平均13分钟才有1次天文事件，错过就得等下个天文事件。
   - 通过建立<a href="?file=001-猫咪百科/01-建筑物/03-科技建筑#天文台">天文台</a>，可以提高出现天文事件的几率。
      - 天文台的提高事件几率可以被玄学<a href="?file=001-猫咪百科/03-科学/02-玄学#时光占卜">时光占卜</a>、<a href="?file=001-猫咪百科/03-科学/02-玄学#占星术">占星术</a>和<a href="?file=001-猫咪百科/06-宗教/003-奥秘神学#耀变体">耀变体</a>加成。
 - 网页版本观测天文事件的按钮在右边日志下面，手机版本观察天文事件的按钮在左下角。
   - 当然得出现了天文事件，**按钮**才能看到。
 - 每级<a href="?file=001-猫咪百科/01-建筑物/03-科技建筑#天文台">天文台</a>加 1% 自动观察的几率。（理解成猫猫有几率自动帮你点击，100%全自动故按钮将不会出现）
  - 工坊升级<a href="?file=001-猫咪百科/04-工坊/01-升级#搜寻地外文明计划">搜寻地外文明计划</a>后，自动观察几率变为100%，按钮将不会出现。
