# 燃烧领导力
<strong>
	 燃烧领导力
</strong>
是一种
<a href="?file=003-资源大全/005-资源介绍#罕见资源">
	罕见资源
</a>。燃烧领导力**重置会保留**。  
1.1.9.1版本更新了燃烧领导力，可以把领导力全部变成燃烧领导力。  
可以让玩家能够完成加星的<a href="?file=001-猫咪百科/09-成就">太空怪谈成就</a>。

## 获取
- 只能在<a href="?file=001-猫咪百科/03-科学/02-玄学">玄学</a>里，通过燃烧领导力按钮，把全部领导力燃烧得到**燃烧领导力**。

## 用途
- 燃烧领导力的直接收益：
 - 每点燃烧领导力提供 1% 的全局生产加成。
   - 1%的全局生产加成
     - 在<a href="?file=005-名词解释/02-黑暗未来">黑暗未来</a>之前，上限为 100% 且加成从 75% 开始<a href="?file=005-名词解释/04-收益递减">收益递减</a>。
     - 在黑暗未来之后，上限为 400% 且加成从 300% 开始收益递减。
     - 这个上限可以通过玄学的卡巴拉树升级来提高。
   - **转化加成**只能**获得**全局生产加成的 5% 收益。（比如影响熔炉，煅烧炉和加速器等等）
     - 即每点领导力提供 0.05% 的转化加成，黑暗未来之前上限为 5%  且加成从 3.75% 开始收益递减。
 - 每点燃烧领导力提供 0.05% 的存储容量加成。
   - 每点燃烧领导力在黑暗未来之前：提高 0.05% 的存储容量（无上限）
   - 每点燃烧领导力在黑暗未来之后：提高 0.2% 的存储容量（无上限）
 - 燃烧领导力可以提升了领袖特质带来加成。
   - 特质加成（%）:$\frac{\sqrt{1+\frac{8\times\text{燃烧领导力} }{500}} -1}{2} $
 -  燃烧领导力还可以降低<a href="?file=001-猫咪百科/08-时间/04-虚空领域#冷冻仓">冷冻仓</a>业 的消耗。<br>
   - 价格降低（%）:$\frac{\sqrt{1+\frac{8\times\text{燃烧领导力} }{500}} -1}{2} $
      - 75%开始<a href="?file=005-名词解释/04-收益递减#有上限收益递减">有上限收益递减</a>，上限为100%。

注:跟领导力的加成叠加，为加法。

## 存储
- 没有限制
