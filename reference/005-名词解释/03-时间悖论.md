# 时间悖论
- 时间悖论是由 [超时空传送仪](?file=001-猫咪百科/01-建筑物/09-超级建筑#超时空传送仪) 引发的事件。
 - 当你超时空传送仪数量 > 0 ，换季节后会自动进入一段日期为负数的时间（最初时长为 10 天），这个时段即是时间悖论。
 - 资源的产出数值 会变成 **???**，也就是该时间里同时大部分资源生产都将停止。
 - 在此期间，只有工程师，AI 核心，纠缠站和时计炉可以正常工作。
 - 时间悖论引起的每个负数天都有机会产出 -1, 0, 1 或 2 [虚空](?file=003-资源大全/53-虚空) （即平均每天产生 0.5 虚空）。
   - 时间悖论的持续时间可被 [时间操纵](?file=001-猫咪百科/08-时间/04-虚空领域#时间操纵) 延长。
   - 虚空单日产出最大值可被 [虚空抽取](?file=001-猫咪百科/08-时间/04-虚空领域#虚空抽取) 提升。


- 理解为抽卡，传送仪的数量为SR的概率，比如1个传送仪 相当于SR的概率是1%，即平均需要100抽才能抽到它。~~
- 第一级传送仪平均进入下次时间悖论需要的季节平均为100

**注意：燃烧 [时间水晶](?file=003-资源大全/20-时间水晶) 跳过的时间不会计入季节变换。**（年份变换）

## 时间悖论概率
- P为百分位，即数字越大越欧（欧皇程度~~非酋~~（%）
- n为传送仪数量
- T为距离下次时间悖论（季节）
- $T =\frac{\ln\_{}{\frac{p}{100} }}{\ln\_{}{\frac{100-n}{100} }}$
- $P=\left ( {\frac{100-n}{100}} \right ) ^{T} $
- 比如距离下次时间悖论需要 69， 1个传送仪，$\left ( {\frac{100-1}{100}} \right ) ^{69} =0.5$就是1个传送仪平均来讲是69
