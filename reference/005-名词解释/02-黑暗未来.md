# 黑暗未来
- 游戏日历年达到 40000 年以后就是黑暗未来。<br><br>
- 黑暗未来有三个影响:
 - **奖励**:
   - 1.<a href="?file=003-资源大全/60-燃烧领导力">燃烧领导力</a>的储存上限加成从0.05% → 0.2%、生产加成上限从100% → 400%。
     - 注意领导力不受影响，储存上限加成0.1%、生产加成上限 200%。
   - 2.日历年超过40K，每 1K 年 <a href="?file=001-猫咪百科/07-空间/11-暗影#HR收割机">HR收割机</a> 的能源产出将提升 1%。
 - **惩罚**:
   - 3.日历年超过40K，每 1K 年 <a href="?file=001-猫咪百科/08-时间/03-时间锻造#燃烧时间水晶">燃烧时间水晶</a> 的价格将增加 1%。
----

- 可以建造<a href="?file=001-猫咪百科/08-时间/03-时间锻造#时间阻抗"> 时间阻抗 </a>**仅**延迟燃烧时间水晶的价格增加，每级时间阻抗可将黑暗未来惩罚到来的时间延长 1K 年。
 - 时间阻抗的效果可以被 <a href="?file=001-猫咪百科/06-宗教/003-奥秘神学">耀变体</a> 和 <a href="?file=001-猫咪百科/08-时间/03-时间锻造#时空加速器">时空加速器</a> 加成。
