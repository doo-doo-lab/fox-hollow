# 挑战
- 购买玄学 <a href="?file=001-猫咪百科/03-科学/02-玄学#仲裁局">仲裁局</a> 将解锁挑战标签。
- 挑战设置了特殊的限制，一旦达成挑战目标，将会获得**基础奖励**（每个挑战只能获得一次）和**另外的重复奖励**。
- 鼠标移动到单独的挑战（进行中）显示的是挑战的负面效果，非进行中显示的是挑战奖励效果<br><br>
- 可以同时挑战你选中的挑战，点击"应用选中挑战"按钮将**重置游戏**并**同时挑战你选中的挑战**，同时正常获得**当前游戏所能获得的领导力和业**。
 - 达成目标之后挑战限制就会解除且会获得奖励，可以继续玩、也可以重置。当然，过程中坚持不下去可以随时重置。
- 如果想同时挑战钢铁意志，可以在选中其他挑战后再点钢铁意志，可以同时选中的挑战们 + 钢铁意志 一起挑战。<br><br>
- 当有传送仪（<a href="?file=001-猫咪百科/08-时间/04-虚空领域#冷冻仓">冷冻仓</a>）后并选中挑战且应用后，**不会携带资源进挑战**，资源（小猫）会被储存起来。
- 在挑战中，无法享受该挑战带来的**奖励效果**。（比如在第二次挑战黑暗天空里，并没有基础奖励里的金字塔效果）
**推荐至少点完玄学文艺复兴再来挑战。**

## 取回储存资源
- 当有资源（小猫）储存过，直到不在挑战中（成功、退出）后才会在挑战标签中出现。
- 点击该按钮会把进入挑战前储存的资源取回给你，包括存着的小猫隔了好几周目也能“复活”。<font size=5>🐱</font>
- **注意**：
  - 只要某周目游戏没有超时空传送仪，保存的资源（小猫）就会消失。
  - 每次挑战前，如果已经储存过资源，那么会计算传送仪效果得到的数量和已经储存资源的数量中的**取最大值**。
  - 对于取回储存资源：
    - 对于**有**储存上限的资源也是取储存和现有资源中的最大值。
    - 对于**无**储存上限的资源是叠加。（木梁等等）
    - 注：时间水晶因为不会储存也就不会获得。

- 在没有小猫的情况下游戏。 
- 没有目标。
- 解锁条件：在没有建造小屋的情况下建造图书馆即可进入钢铁意志模式。
  - 进入后营火 建筑标签:可切换右边会出现IW(IronWill缩写)
- 奖励：无  (各种成就)
- 说明：
 - 当独角兽小于2每天有0.035%几率到来一只独角兽。（欧皇在造猫薄荷田的时候已经到来过独角兽的原因）
 - 钢铁意志的一周目，最多只有一只斑马猎人。
  - 如果科技完成了箭术，且你的斑马猎人数量上限小于10，则会立即到来一只斑马猎人。
  - 斑马猎人数量还未到达上限时，每天有0.5%几率到来一只斑马。（即显示xx/xx才会有斑马猎人到来）
    - 每研究10k科学（仅工坊升级和科学），重置后的下一周目斑马猎人数量上限将增加1，每轮游戏最多达到100。
    - 如果有玄学斑马外交、斑马盟约，部分斑马会保留到下周目，但会以上限的形式增加。
    - 斑马猎人不受幸福度加成
 <!--- 离开IW会立刻失去所有斑马猎人。-->
 - 天文事件有基础25%概率自动观测，天体力学的奖励从20%提高到60%。
 - 工坊[升级板块]中的：复合弓（200），投石索（400），狩猎盔甲（1000）可以使基础喵力上限提升，最多为1000。（基础喵力上限100所以最高是900）
 - 喵力获取速度会随着斑马猎人数量超过10个后收益递减。
 - 流星获得的矿物产量+10%。
 - 仅建筑的矿物加成可以增加流星获得 矿物 数量。

   - 比如政策皆伐的没有效果
   - Tips: 仅营火建筑的科学加成可以增加流星和天文事件获得 科学 数量(非钢铁意志也是)

 - 大使馆的效果只有原来的1/4。

- 收益：★☆☆☆☆
- 重复挑战收益：★☆☆☆☆<br><br>
注：在冬天来了挑战中饿死于**寒冬**的小猫(即文本会是冻死~)是不会再有新的小猫加入的

### 无政府状态
- 描述：重置游戏后，猫咪会按自己的方式生活，变得懒惰，消耗额外的猫薄荷，并且无法被设置成领袖。
- 目标：建造<a href="?file=001-猫咪百科/01-建筑物/09-超级建筑#AI核心">AI核心</a>。<br><br>
- 基础挑战：猫咪不能被设置成领袖，猫咪不再获得经验。有一部分小猫会变得好吃懒做，基础慵懒度为50%，即一半的猫猫将不会工作，且消耗额外的猫薄荷。
- 基础奖励：业力分段计算的时候按照双倍的猫咪数量计算。<br><br>
- 重复挑战：从第一次挑战以后开始，每进行一次挑战，会增加额外5%的慵懒度，上限75%即从56%开始递减。
- 重复奖励：每完成一次挑战，猫咪从大师的职业熟练度获得的工作效率额外提高20%，上限400即从300%开始递减。
 - 大师的职业熟练度基数18.75%，即每次挑战成功提高大师的职业熟练度提高3.75%。<br><br>
- 难度：★★☆☆☆
- 收益：★★★★★
- 重复挑战收益：★☆☆☆☆<br><br>

**注**：
- **当前不工作的猫咪数量（懒惰猫咪） = 猫口上限 \* 慵懒度（向上取整）**
- 所有猫咪都会消耗工作时的猫薄荷，即每秒猫薄荷消耗 = 基础消耗（4.25）\* 幸福度加成
- 建筑都是默认隐藏的，即看不到AI核心，是因为没反物质，需要<a href="?file=001-猫咪百科/07-空间/07-太阳#太阳提取">太阳提取</a>产出。

### 千禧年
- 描述：重置游戏后，进入半衰期时间。
- 解锁要求：<a href="?file=001-猫咪百科/08-时间/03-时间锻造#燃烧时间水晶">燃烧过至少一次时间水晶</a>。（需工坊升级<a href="?file=001-猫咪百科/04-工坊/01-升级#时间锻造">时间锻造</a>）
- 目标：达到1000年。<br><br>
- 基础挑战：**500年后，年份将不再自然增长**。
- 基础奖励：燃烧时间水晶只会产生一半的时间热（译注：从10变成5）。<br><br>
- 重复挑战：从第一次挑战以后开始，每进行一次挑战，燃烧1个时间水晶价格会涨价 0.5 时间水晶和 0.4 虚空。
- 重复奖励：每完成一次挑战，燃烧时间水晶的消耗都会减少2%，会<a href="?file=005-名词解释/04-收益递减#有上限收益递减">收益递减</a>。
<br>时计炉散热获得热能量提高 10%,燃烧时间水晶所产生的热度会减少5%<br><br>
- 难度：★☆☆☆☆
- 收益：★★★★★
- 重复挑战收益：☆☆☆☆☆<br><br>
在半衰期年间（500年后）资源会涨、周期、季节都会变，仅年份固定不涨（统计里的时间会固定）。**只能燃烧时间水晶到达1000年**。  
第一次做比较简单，只要有足够高的太阳革命，可以0水晶进入挑战当场贸易或者献祭天角兽即可。  
由于虚空只能是整数，推荐以 5 的倍数燃烧时间水晶。（燃烧 0.4 虚空 会进一直接减少 1 虚空）  
第二次挑战成功，解锁<a href="?file=001-猫咪百科/08-时间/03-时间锻造#时空增压机">时空增压机</a>的自动化。
燃烧时间水晶产生热度: 5 / (1 + 0.05 x 千禧年次数)

### 和平主义
- 描述：重置后不再能打猎，一部分政策不再能选择。
- 解锁要求：一次性打猎1000次以上。
- 目标：选择 <a href="?file=001-猫咪百科/03-科学/03-政策#外层空间条约">外层空间条约</a> 政策后重置<br><br>
- 基础挑战：**不能打猎，一部分政策不再能选择**。
- 基础奖励：每个交易所可以增加0.099%的贸易比率。<br><br>
- 重复挑战：从第一次挑战以后开始，每进行一次挑战，<br>大使馆的价格额外 x 1.15，政策的价格额外 x 1.25，蒸汽工房蓝图初始价格额外增加5，蒸汽工房其他材料的价格额外 x 2。
- 重复奖励：每完成一次挑战，天角兽降临的概率会提高10%（上限为120%）、天角兽产量会提高10%、贸易关系会增加1。<br><br>
- 难度：★★★☆☆
- 收益：★★★☆☆
- 重复挑战收益：★★☆☆☆<br><br>
