# 玄学
<font color=red size=4>因为没有确认弹窗，千万别点错了。</font>~~没有后悔药哦~~

- 研究完科学里的<a href="?file=001-猫咪百科/03-科学/01-科学#玄学">玄学</a>后，并且拥有2个<a href="?file=003-资源大全/50-领导力#领导力">领导力</a>后解锁玄学板块。
- 玄学里面的解锁完升级效果就是**立即生效**且永久存在，重置后里面的启蒙运动等等升级效果也**会生效**。
  - 但玄学本身这个科技重置不继承，也就是每周目如果需要解锁新的“玄学”科技，都要花 5难得素 重新研究。
- 玄学减少的价格比仅影响<strong>营火</strong>标签。

注意，如果想查看以解锁的玄学，只用点击科学标签后，最上面右边有个不勾选隐藏已完成的科技即可。

- **领导力和燃烧领导力**的**当前总和**大于临界点的数字，即该卡巴拉树玄学后库存效果才会比不点该卡巴拉树玄学的高。

  - 注后期领导力的库存为首要，**推荐领导力数字大于临界点在点卡巴拉玄学**。

### 卡巴拉：王国（Malkuth）
- 领导力的效能：领导力或燃烧领导力生产加成提高5%，库存上限提高5%。
- 即生产加成和库存加成 \* (1 + 效能总和)。
  - 比如点到了卡巴拉：王国（Malkuth），领导力的生产加成：196 ⇒ 196 \* (1 +5%) = 205.8，
  - 比如点到了卡巴拉：基础（Yesod），燃烧领导力的生产加成：96 ⇒ 96 \*  (1 + 10%) = 105.6%，
