# 村庄与职业

- 小猫可以分配工作，以获得 <a href="?file=003-资源大全/005-资源介绍">资源</a>，如 <a href="?file=003-资源大全/03-矿物">矿物</a>，<a href="?file=003-资源大全/14-科学">科学</a> 和 <a href="?file=003-资源大全/15-文化">文化</a>。
- 分配作业的小猫越多，资源的收集速度就越快。
 - 注意：升级完 <a href="?file=001-猫咪百科/04-工坊/01-升级#强化">后勤</a>后，可以提高对应小猫 的 职业 的 熟练度级别 的额外工作效率 提高15%。  
 - 升级完<a href="?file=001-猫咪百科/04-工坊/01-升级#强化">强化</a>后，可以提高对应小猫 的 职业 的 熟练度级别 的额外工作效率 提高100%。
