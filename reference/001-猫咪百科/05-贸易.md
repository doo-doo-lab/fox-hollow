# 贸易
- 至少需要发现任意一个种族才能解锁贸易标签。
- 显示的数字是会自动四舍五入转化成整数。
  - 出售的数量 上下浮动 **20%** ，符合二项式分布，即数值越多越接近理论值。
  - 出售的数量 可以被**贸易所得**和政策加成。
  - 游戏运行会把所有小数计算进去，所以可能会有些许误差。
- *小猫空爪而回* 对应获取不到资源，比如原因获得资源已经上限了等等。
- 派遣探险队花费 1000 喵力可以找到新的种族，如果你达到了条件，就能发现这个种族。
- 派遣探险队如果没有发现的新的种族将会退还 900 喵力，并有提示下个种族的条件。<br><br>

## 种族地位
- 敌对的文明贸易，小猫有几率空爪而归。（“ <em>[种族]</em> 无缘无故恨你”）
 - 斑马初始 **-30**% ：狮鹫初始 **-15**%。
 - 只有敌对可以改善成中立。
- 友好的文明贸易，小猫有几率获得额外25%的出售资源。（“ <em>[种族]</em> 认为你的小猫非常可爱”）。
 - 蜥蜴初始 +25% ：蜘蛛初始 +15%。
 - 友好事件触发最高100%概率
- 改善关系**仅**有在工坊<a href="?file=001-猫咪百科/04-工坊/01-升级#商队驿站">商队驿站</a>后的<a href="?file=001-猫咪百科/01-建筑物/08-其它建筑#交易所">**交易所**</a>、某些政策和玄学。（注意<a href="?file=001-猫咪百科/05-贸易#大使馆">大使馆</a>不能改善地位）
  - 每个交易所将敌对事件发生的概率降低 0.35%。
   - 比如 30 / 0.35 ≈ 86个交易所将会让斑马不再仇恨你
   - 每个交易所将友好事件触发概率增加 0.175%。
 - **政策**相当于提供（**x个**）交易所的效果:
   - <a href="?file=000-猫国萌新指导/02-政策效果解析#内事">自由贸易</a>（+10）、<a href="?file=000-猫国萌新指导/02-政策效果解析#外事">外层空间条约</a>（+10）、<br><a href="?file=000-猫国萌新指导/02-政策效果解析#外事">斑马外交：绥靖政策</a>（对于斑马种族 +15）<br><a href="?file=000-猫国萌新指导/02-政策效果解析#外事">斑马外交：剑拔弩张</a>（对于斑马种族 **-10** : 非斑马种族 +5）
 -  玄学<a href="?file=001-猫咪百科/03-科学/02-玄学#外交手段">外交手段</a>相当于 29 个交易所。

## 附加说明

### 斑马
- **获得钛的机会**（%）：0.35 × <a href="?file=001-猫咪百科/04-工坊/02-工艺#贸易船">贸易船</a>的数量 + 15
 - 因此，需要242.858个贸易船才能 100% 获得钛（需要成功交易）。
- **获得钛的数量**：0.03 × 贸易船数量 + 1.5
 - 钛的数量可以被斑马外交：绥靖政策加成22.5%，或者被斑马外交：剑拔弩张减少22.5%。
- 在所有季节，获得钛的机会 和 数量 都相同
- 注意贸易获得的资源，只有**钛**的数量不会受到贸易所得的加成。
  - 钛只与贸易船数量有关，参考工坊-工艺制作-贸易船-描述。且贸易船**只**会增加**钛**的贸易数量。

#### 利维坦献祭
<ul>
    <li>如果你已经建立了一个或多个<a href="?file=001-猫咪百科/06-宗教/001-庙塔#神印">神印</a>，并且至少有一个天角兽，你会慢慢地产生<a href="?file=003-资源大全/47-死灵兽">死灵兽</a>。死灵兽可以喂给利维坦来增加能量并获得额外的好处。</li>
    <li>在贸易标签中，利维坦有一个“献祭死灵兽”按钮。单击此按钮的工作原理如下：</li>
    <ul>
        <li> 如果你有死灵兽，上古神会很高兴，他们的能量将会增加1。</li>
        <li> 如果你没有死灵兽，上古神会生气，他们会立即离开，从贸易窗口消失。</li>
    </ul>
</ul>

#### 利维坦能量
<ul>
    <li> 在游戏一开始或重置后，利维坦的能量均为0。每献祭利维坦一个死灵兽，能量增加1。</li>
    <li> 能量有如下效果：</li>
    <ul>
        <li> 每点能量延长利维坦贸易前的停留时间1年，停留时间为（5+能量点数）年。</li>
        <li> 每点能量使利维坦在贸易后停留更长的时间，停留时间为（1 + 0.25 \* 能量点数）年。 </li>
        <li> 每点能量与利维坦在贸易中可以获得+ 2%的贸易资源数量加成。</li>
        <li> 从1.4.1.0开始，利维坦的能量上限为 5 + 5 × 神印数量。</li>
        <ul>
            <li> 利维坦能量上限还可被政策死灵统治和挑战黑暗天空的完成次数加成。(四舍五入取整)</li>
        </ul>
    </ul>
</ul>

#### 惹怒看不见的黑手
<ul>
    <li> 在工坊<a href="?file=001-猫咪百科/04-工坊/01-升级#看不见的黑手">看不见的黑手</a>升级完后，利维坦贸易界面会出现“惹怒看不见的黑手”的按钮。</li>
    <li> 点击“惹怒看不见的黑手”的按钮作用：</li>
    <ul>
        <li>消耗一定数量的时间水晶，则会使的黑币价格下降0.3 - 0.1 \* [0~1]之间的随机数</li>
        <li>当黑币价格低于550时，惹怒看不见的黑手将不可点击。</li>
    </ul>
    <li> “惹怒看不见的黑手”的时间水晶价格是根据多方面有关具体公式为：</li>
    <ul>
        <li> 游戏内公式是以前写的且没再更新，所以政策方面增加难得素或者贸易所得都不会影响惹怒看不见黑手的价格。<br>(注意下面的加成都是默认，比如你的领袖特质不是商人也是会算商人加成的)</li>
        <li> 一年难得素产量都跟利维坦贸易获得水晶数量 N = 空间生产加成 \* 太空制造加成 \* 交易所的贸易加成 \* 商人领袖的贸易加成 \*<br> 利维坦能量加成 \* 游戏1年的总tick次数 \* 微型亚空间反应装置加成 \* 基础月球前哨每秒产生的难得素 \* 0.25 \* 0.98 / 5000 (利维坦贸易)</li>
        <li> 也就是只有月球前哨、太空电梯、轨道阵列、工厂、交易所的数量有关。（想降低价格只能出售，关闭是没有作用的）</li>
        <li> A = 1.13 (平均50周期难得素产量的系数) \* N / 4000， B = 2.4 (红月年的难得素系数) \* N / 4000， C = A + 1.13 \* （资源回复 \* 耀变体的加成）\* 时计炉系数</li>
        <li> D =（在A，B，C中取最大一个数 ）\* 8000466.693057134 \* ln(1100 / 当前黑币价格) </li>
        <li> 然后在 D 和 256 中取最大一个数为E。
        <li> x = lnE / ln2。（即x最小为8）
        <li> 时间水晶价格为2的x次方，x需要向上取整。</li>
    </ul>
</ul>
