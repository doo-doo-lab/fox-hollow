# 不重置猛男之从看懂到重置
看似重置会全部重来，但因为有领导力和业的奖励，重置会比不重置快10倍的速度。<br>
毕竟重置相当于一个功能，少用这个功能发展就会慢很多<br>
**重置的可以参考萌新指导**
**重置的正常2个月毕业。不重置24小时挂的需要20月以上**。  
推荐重置档毕业后可以再开档不重置，或者同时玩2个档在不同的网站。<br>
**政策点错的建议直接重来**，发展差距会很大<br>

可以参考游戏标签了解效果。  
流程简要 5000年（5点领导力） ⇒ 点启蒙运动 ⇒ 凑5000反物质解锁<a href="?file=001-猫咪百科/04-工坊/01-升级#遗物站" target="_Blank">遗物站</a> ⇒ 次元超越15 ⇒ 15000年（狂妄自大）⇒ 就开始时间水晶的自由 ⇒<br> 充足的时间水晶 燃烧到 20M年 ⇒ 8G难得素 100传送仪。  
**推荐礼物盒**：文艺复兴(仅减价玄学黄金比例、神圣比例、维特鲁威猫)或者1500燃烧领导力<br>

### 营火
- 对于营火就有上限就造到上限，没上限造到你喜欢的数字即可。
- 推荐最后关头造收容室差科学上限再造AI核心，确保AI等级别到15级
  - 推荐4000年左右开始造

### 政策
由于不重置，对应的政策就需要些许改变，注意政策选择后当周目无法改变。（**如果真要挂40K年，政策点错建议直接重来，挂的时间越久政策影响越大**）
- <a href="?file=000-猫国萌新指导/02-政策效果解析#内事">内事</a>：
 - 由于时间水晶主要跟利维坦贸易，也就是必定要点自由贸易，于是政策路线：**自由** ⇒ **民主** ⇒ **自由贸易**。
 - 显然为了难得素和库存，故选择 **新古典宇宙自由主义** 和 **超猫主义**。
- <a href="?file=000-猫国萌新指导/02-政策效果解析#外事">外事</a>：
 - 由于后期黄金产出会远大于喵力，故选择 **外交**。（每次贸易消耗的喵力 -5）
 - 科学比文化重要点，故选择 **知识共享**。（科学产出增加5%。）
 - 由于时间水晶主要跟利维坦贸易，但斑马外交：剑拔弩张会减弱斑马贸易的钛、铁数量，所以一开始**不点**<br>等能与利维坦贸易时再选择**斑马外交：剑拔弩张**
 - 由于时间水晶主要跟利维坦贸易，故选择 **外层空间条约**。
- <a href="?file=000-猫国萌新指导/02-政策效果解析#哲学">哲学</a>：
 - 奢侈品路线的不难抉择：**伊比鸠鲁主义** ⇒ **奢侈**。
 - 由于不重置，没有领导力的情况下煤会比铁缺点，所以理性看起来没那么的好，故选择 **神秘**。
- <a href="?file=000-猫国萌新指导/02-政策效果解析#环境">环境</a>：
 - 木材前期比较缺，故选择 **皆伐**。
 - 一开始工厂升级都没解锁，所以一开始**不点**<br>等点出<a href="?file=001-猫咪百科/04-工坊/01-升级#太空制造">太空制造</a> 再选择 **全面工业化**。
   - 全面工业化加成难得素产量，也就是你的时间水晶或者领导力~

### 空间
- 在未到达15000年**少造月球前哨、不造空间站和轨道阵列**，
  - 难得素需求没那么高，空间站和轨道阵列是耗电
- 15000年前缺电可以造完月球基地(或者各种加上限的建筑)，提高上限后造建筑，造完再把月球基地关了 

#### 遗物站
- 主要是靠AI核心凑够17.2M科学上限造够30收容室
  - 缺科学上限的时候可以造AI核心增加数据中心的效果(需要工坊升级<a href="?file=001-猫咪百科/04-工坊/01-升级#机器学习" target="_Blank">机器学习</a>)
  - 等快到4000反物质上限在造AI核心
    - 千万别让AI核心等级到达15级!!!
- 凑够5000反物质上限会缺电，所以有以下对应措施：
  - **关闭生物实验室（- 全部），关闭加速器（- 全部）**。
    - 加速器、生物实验室均不影响**库存**。
  - 关闭**油井自动化**
  - 关闭**空间站**
  - 关闭**工厂自动化**
  - **其他可选**关闭煅烧炉、月球前哨,记得打开反应堆自动化
- 注意反物质上限低于5000时，每低于1点，遗物将减产0.02%。
- 群文件我有上传5000反物质上限 需要的 收容室和散热器的数量excel。
- 研究完遗物站，推荐憋出一个纠缠站

### 太阳教团
- 次元超越等级是一周目不重置的核心
- 上了太空可以把多余的地质学家分配到牧师(比如先减到35个在慢慢减)
- 由于没领导力，太阳革命前面最多有1000%的加成，所以未来首要发展太阳教团。
- 推荐解锁新约外传后可以攒满信仰 ⇒ 赞美群星 ⇒ 赞美太阳。(再150K虔诚左右凑0.25的顿悟)
- 推荐解锁超越后 攒满信仰 ⇒ 根据概览推荐次元超越 如果“是” 就次元超越（一般可以几次）⇒ 赞美群星 ⇒ 赞美太阳。
- 虽然赞美群星会重置虔诚，但赞美群星和次元超越是获得更高的太阳革命加成。
 - 因为只有次元超越等级高，才能到达900%多的太阳革命加成。
 - 后续奥秘神学的解锁需要次元超越等级。非常需要15次元超越解锁的<a href="?file=001-猫咪百科/06-宗教/003-奥秘神学#耀变体">耀变体</a>
- 或者就交给珂学家自动就行~
- 次元超越是后面的根本越高越好。（太阳革命、奥秘神学、契约都有关）
