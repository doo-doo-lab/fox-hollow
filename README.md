# 小狐大世界

### Fox Hollow Tales

一群狐狸，一座山谷，一段从野莓开始的文明史。

---

白底黑字，没有图标，没有动画。你看到的只有文字——
但文字底下，有一整座山谷在呼吸。

早春的莓果园开始返青，盛夏的灵狐祠石阶被晒得发烫，
金秋的种子飞过篱笆落进未知的树林，
寒冬的月光井里，雪花落进幽深如墨的水面，然后被吞没。

这是一个增量放置游戏。
也是一个关于"留下"和"出发"的故事。

---

## 山谷里发生了什么

你带一群狐狸，从摘野莓开始过日子。砍木头，挖石头，搭窝棚。慢慢有人（有狐？）搬进来，村子越来越大，开始琢磨怎么把石头磨成工具，怎么用木头搭房子，怎么从矿石里弄出铁。

然后你会听说：山外面还有别的地方。

你可以派斥候出去看看。他们会带回来石头、兽皮、古币、香料，还有故事——关于一座废弃的狐狸城邦，关于云上面那条路，关于几百年前某只狐狸在一块石头上给陌生狐狸留下的字。

不着急，慢慢玩。玩着玩着就都知道了。

---

## 游戏里有什么

资源：野莓、圆木、碎石、符咒、古币……一共十二种，从地上捡的到祖先留的。

建筑：莓果园、狐狸窝、伐木场、月光井、瞭望塔……十四种，一点点把山谷填满。

职业：采集的、砍树的、挖矿的、打猎的、看书的、打铁的、做生意的，还有往外跑的斥候。

研究：从“把石头敲出刃”到“走远路不回头”，十一项。

灵术：祈雨、祖灵、山谷宴席……六种，多数是求个心安，偶尔真管用。

远行：四个地方能去，近丘、密林、旧墟、云岭。去一趟带点东西回来，也带回一些故事碎片，慢慢拼。

商队：山猫、河獭、白鹤、旧墟来的灰毛狐狸。他们卖的东西不一样，告别的方式也不一样。

季节：早春东西多，冬天冷得要死。四季转一圈十二分钟。

## 技术

- 纯前端：HTML + CSS + JavaScript，无框架，无依赖
- 白底黑字纯文本界面
- localStorage 自动存档 + Base64 存档码跨设备迁移
- 离线补算（最长 24 小时）

```
fox-hollow/
├── index.html
├── css/style.css
├── js/
│   ├── data.js      # 数据定义
│   ├── engine.js    # 游戏引擎
│   └── ui.js        # 界面渲染
└── docs/
    └── design.md    # 策划文档
```

## 开始游戏

用浏览器打开 `index.html`，或访问线上版本：

**https://akira17189-create.github.io/fox-hollow/**

点击「采集野莓」。剩下的事，山谷会慢慢告诉你。
