# 神启副线详细设计

> ⚠️ **权威性说明**：本文档为神启副线的概览文档。Phase B-D 的详细数据（研究 ID、建筑参数、配方费用、升级效果）以子文档为准：
> - 教团路：[branch-divine-doctrine.md](branch-divine-doctrine.md)
> - 秘仪路：[branch-divine-mystery.md](branch-divine-mystery.md)
> 本文档的 Phase A 详细设计（§四）为权威版本，Phase B-D 仅为摘要。

> 副线定位：叠加在任意主线之上的信仰系统，体量约为主线的 70%。
> Phase A 共通奠基；Phase B 根据主线自动分叉为**教团**（工业+神启）或**秘仪**（灵修+神启）；Phase C 深化展开；Phase D 各自终局。
> 教团 = 组织化宗教狂热驱动建造（中世纪教廷）；秘仪 = 个人化密教修行走向飞升（密教模拟器 × 古希腊秘仪）。

## 一、与主线的关系

<!-- SECTION: main-branch-integration -->

### 核心原则：副线是加法，分叉靠主线自动决定

| 组合 | Phase B 路线 | 效果 | 终局 |
|------|-------------|------|------|
| 工业 + 神启 | **教团** | 工业产能 + 神恩加成 + 教令系统（限时buff） | 太空 + 神佑加成（不改终局类型） |
| 灵修 + 神启 | **秘仪** | 灵修循环 + 神恩加成 + 飞升阶梯（永久蜕变） | **神升**（替代灵迁） |
| 纯工业/纯灵修 | — | 无神恩加成 | 太空/灵迁 |

### 净收益保证

- 神恩加成（+3%~50% 全局产出，升级可至 80%）远大于投入成本
- **教团路无负面惩罚**，纯机会成本（资源投入换临时buff）
- **秘仪路有主动代价**（第四、五重门会损失人口上限等），但收益远大于代价
- 总体设计：开副线的玩家比不开的**明显更强**

## 二、神恩系统（共通核心机制）

<!-- SECTION: divine-grace -->

### 定位

对标猫国太阳革命（Solar Revolution）：虔诚越多 → 全局产出加成越高。
Phase A 共通，两条路共享同一套神恩公式。

### 公式

```
graceBonus = min(graceCap, (sqrt(1 + piety/100) - 1) × 50%)
```

- 默认上限 `graceCap = 50%`（升级可推至 80%）
- 需研究「恩典感召」后生效

| 虔诚存量 | 神恩加成 | 备注 |
|---------|---------|------|
| 0 | 0% | — |
| 25 | ~7% | Phase A 早期 |
| 100 | ~20% | Phase A 满配典型值 |
| 400 | ~36% | Phase B 中期 |
| 900 | ~50% | 默认上限 |
| 1600+ | ~64%→80% | 需升级提升上限 |

### 引擎集成

- 在 `calcR()` 中，全局乘数应用之后插入：
  ```js
  // 神恩全局产出加成
  if (G.upg.graceLore?.done && G.res.piety) {
    var p = G.res.piety.v;
    var graceBonus = Math.min(G._graceCap || 0.50,
      (Math.sqrt(1 + p / 100) - 1) * 0.50);
    if (graceBonus > 0)
      for (const k of Object.keys(r)) if (r[k] > 0) r[k] *= (1 + graceBonus);
    G._graceBonus = graceBonus; // 缓存供 UI 显示
  }
  ```
- `G._graceCap` 默认 0.50，升级可修改
- 仅作用于**正向产出**（`r[k] > 0`），不放大负向消耗

### 与主线系统的隔离

- 不影响灵脉/能量计算（神恩乘数在 calcLeyline/calcEnergy 之后）
- 不影响污染/躁念计算（独立系统）
- 仅作用于资源产出速率

## 四、Phase A 详细设计（共通）

### 步骤 1：神恩系统

见第二节。实现要点：
- `G._graceBonus`：当前神恩加成值（缓存，每 tick 更新）
- `G._graceCap`：神恩上限（默认 0.50，升级可提升）
- `resetG()`：初始化 `G._graceBonus = 0; G._graceCap = 0.50;`
- `migrate()`：兜底 `G._graceBonus = G._graceBonus || 0; G._graceCap = G._graceCap ?? 0.50;`
- UI：虔诚资源 tooltip 中显示当前神恩加成百分比

### 步骤 2：资源

**data.js RD 新增：**
```js
// ===== 神启副线 A阶段：初悟 =====
piety: { n: '虔诚', c: '研究', mx: 0, lock: 1 },
holyOil: { n: '圣油', c: '研究', mx: 25, lock: 1 },
```

- 全部使用共享资源（符咒、碎石、卷轴、木板、虔诚）
- 不使用工业资源（煤/钢/火油）或灵修资源（灵能/命丝）

## 六、Phase B-秘仪：神露与飞升（灵修+神启）

> **一句话**：你不是在修炼灵术，你是在**一步步变成神**。

### 飞升阶梯（秘仪独占机制）

#### 定位

五重门 = 五次不可逆的永久蜕变。每扇门需要大量秘知+神露，打开后永久改变角色状态。越接近飞升，代价越大（放弃凡俗的东西）。核心决策：**什么时候攒够资源开门？是否愿意承受后期代价？**

#### 五重门

| 阶 | 门名 | 消耗 | 永久效果 | 永久代价 | 风味 |

#### 飞升规则

- 必须按顺序开门（一→二→三→四→五）
- 每扇门不可逆，开了就回不去

- 第五门直接触发终局条件（但玩家仍可选择不立即触发终局事件）
- 化神池建筑等级 = 可开门的最高阶数（化神池×1 = 可开第一门，×3 = 可开第三门，×5 = 可开第五门）
