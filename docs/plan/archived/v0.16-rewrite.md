# v0.16 重写方案：分层不可逆路线树（定稿）

> 基于 v0.16-plan.md §十三 重构提案 + 2026-04-30 决议。
>
> **核心变更**：
> 1. 取消「议簿/议事录」专用资源 → 改用现有通用资源（卷轴+铜钱+古币）解锁各 Tier
> 2. 资源命名：「谷声」（纯累积门槛型进度指标，不消耗）
> 3. 经济模型参照猫国：实际花费走通用资源，天然有机会成本

---

## 〇、设计哲学：为什么参照猫国

猫国政策系统的核心智慧：

1. **政策花费挂在通用资源上**（文化 → 同时用于手稿、概要、节日、政策），机会成本天然存在
2. **不可逆 + 高门槛**，选错认栽，下一轮重来
3. **层级递进**，早期选择锁定后期路线
4. **每层解锁有明确的资源竞争**（"1500 文化花在政策还是手稿？"）

本方案完整移植这套逻辑到狐狸谷：
- 猫国的「文化」→ 狐狸谷的「卷轴 + 铜钱」（已有多种竞争消耗）
- 猫国的「传统」进度条 → 狐狸谷的「谷声」累积门槛
- 猫国的不可逆政策树 → 狐狸谷的 Tier 1→2→3 级联路线

## 一、「谷声」—— 累积门槛型进度指标

### 1.1 定位

「谷声」不是花费型资源，而是**累积型进度指标**（类似猫国的「信仰」上限解锁太阳能力，但更简单）。

- 攒到阈值 → 解锁对应 Tier 的选择权
- **永不消耗、永不减少**
- 真正的花费走卷轴/铜钱/古币

### 1.2 数据定义

```js
// RD 中
valleyVoice: { n: '谷声', c: '知识', mx: 0, lock: 1, 
  tip: ['所有说出口的话都留在了谷底，叠成了一种不再散去的声音。'] }
```

### 1.3 来源

| 来源 | 产量 | 条件 |
|------|------|------|
| 每年开年 | +5 | 自动（`G.season === 0 && G.day === 0`） |
| 议事堂被动 | +0.001/s/座 | 持续 |
| 激活习俗 | +3/个（一次性） | 激活时发放 |
| 完成远行 | +2/次 | 返回时发放 |
| 完成研究 | +1/次 | 研究完成时 |

### 1.4 产量预估

假设玩家 v0.16 解锁时已有：议事堂 2 座、10 习俗已激活、远行若干。

- 年产：开年 5 + 议事堂 2 座 × 0.001/s × 4000s/年 ≈ 5 + 8 = **~13/年**
- 一次性：10 习俗 × 3 = 30（解锁时已有）
- 远行：约 4 次/年 × 2 = 8/年

**综合年产约 21，起步已有 ~30。**

### 1.5 各 Tier 门槛

| Tier | 谷声门槛 | 预计达到时间 | 实际花费（通用资源） |
|------|----------|-------------|---------------------|
| Tier 1（基本路线）| ≥ 30 | 解锁即可选 | 卷轴 30 + 铜钱 30 |
| Tier 2（政体）| ≥ 60 | ~2 年后 | 卷轴 80 + 铜钱 80 + 古币 10 |
| Tier 3（政策各域）| ≥ 100 | ~4 年后 | 卷轴 50 + 铜钱 40 每域 |

### 1.6 机会成本分析

v0.16 阶段玩家卷轴/铜钱的竞争消耗：

| 消耗场景 | 卷轴 | 铜钱 |
|----------|------|------|
| 研究（剩余未完成） | 30-50 | — |
| 工坊制作（抄录卷轴本身消耗学识+兽皮） | 间接 | — |
| 授业升级 | — | 5-15/次 |
| 商队购买香料/丝帛 | — | 20-50/次 |
| 建筑造价（含铜钱项） | — | 20-50 |
| **Tier 1 路线选择** | **30** | **30** |
| **Tier 2 政体选择** | **80** | **80** |

选 Tier 2 花 80 卷轴 = 放弃约 4 个研究 或 80 天的学者产出。这是真痛。

## 二、取消专用资源后的建筑重新定位

### 2.1「占位：议事堂」→ 谷声产出 + 学识加成

```js

  t: 'b',
  d: '谷中议事之所。每座持续积累谷声，并提升学识产出。',
  p: [
    { r: 'plank', b: 15, k: 1.12 },
    { r: 'brick', b: 10, k: 1.12 },
    { r: 'wood', b: 50, k: 1.12 },
    { r: 'coin', b: 20, k: 1.12 }
```

- 谷声 +0.001/s/座（年产 ~4/座）
- 学识 +3%/座（加法叠加，对标月光井 +3%/座全资源）
- 玩家建 3 座 → 年产谷声 +12，学识 +9%


- 解锁条件：已选定 Tier 2 政体

## 六、Tier 3 —— 政策域（各域 3 选 1，受政体级联控制）

### 解锁条件
- 研究「活络法」完成
- 谷声 ≥ 100
- Tier 2 已选定

### 选择花费（每域独立）
- 默认：**卷轴 50 + 铜钱 40** 每域
- 合嗓令政体：免费（penalty 代偿）

### 不可逆规则
- **默认不可逆**（当期不可改，直到轮回/重置机制上线）
- **守尾制额外惩罚**：选定即永锁（连轮回都不重置——设计意图让玩家极度谨慎）
- **合嗓令**：可逆（不锁定），但数值 ×0.5

### 级联锁定表

| 政体 | 可选域 | 锁定/禁用 |
|------|--------|-----------|
| 守尾制 | 地利、序位 | 外通→锁定为「有度可循」，传习→锁定为「口耳相传」 |
| 闭谷修灵 | 传习、序位 | 外通→锁定为「闭谷自足」，地利→锁定为「合犁同收」 |
| 合嗓令 | 四域全开 | 无锁定（数值 ×0.5） |
| 通货集 | 外通、地利 | 外通禁选「闭谷自足」，序位→锁定为「能者居先」 |
| 野风志 | 地利、外通 | 序位→锁定为「循齿而序」，传习→锁定为「各悟其道」 |
| 散爪活 | 仅地利 | 传习/外通/序位全禁用 |

### 9.1 calcR() 中政体效果

```js
// Tier 1
if (G.tier1 === 'inner') {
  for (var k in r) if (r[k] > 0) r[k] *= 1.08;
}

// Tier 2 政体
if (G.polity) {
  var pe = POLITY[G.polity].e;
  var boost = 1 + Math.min(3, G.bld.polityHall?.c || 0) * 0.05; // 令台上限 3 座
  for (var ek in pe) {
    if (ek.endsWith('M') && pe[ek] > 0) {
      // 正面效果受令台加成
      m[ek.slice(0, -1)] = (m[ek.slice(0, -1)] || 1) + pe[ek] * boost;
    } else if (ek.endsWith('M') && pe[ek] < 0) {
      // 负面效果不受令台加成
      m[ek.slice(0, -1)] = (m[ek.slice(0, -1)] || 1) + pe[ek];
    }
  }
}

// Tier 3 政策
if (G.policies) {
  var policyMul = (G.polity === 'harmony') ? 0.5 : 1; // 合嗓令减半
  for (var dom in G.policies) {
    var optId = G.policies[dom];
    if (!optId || !POLICY[dom]?.opts[optId]) continue;
    var pe = POLICY[dom].opts[optId].e;
    for (var ek in pe) {
      if (ek.endsWith('M')) {
        m[ek.slice(0, -1)] = (m[ek.slice(0, -1)] || 1) + pe[ek] * policyMul;
      }
    }
  }
}
```

### 10.2 通用资源供需压力

**Tier 1 + Tier 2 + Tier 3 全部选完总花费：**
- 卷轴：30 + 80 + 50×2~4 = **210~310**
- 铜钱：30 + 80 + 40×2~4 = **190~270**
- 古币：10

**v0.16 阶段卷轴产量估算**：
- 学者 5 人 × 0.003/s × 4000s/年 ≈ 60/年（含培训/满意度/专精）
- 藏书阁 3 座 × 0.005/s = 额外 ~60/年
- 远行 + 抄录工坊 ≈ 10-20/年
- **年产约 130-140 卷轴**

选完全部 Tier 花 ~250 卷轴 ≈ 2 年卷轴产出。期间研究、抄录卷轴（需学识+兽皮）等正常消耗也在抢卷轴。

**结论**：选 Tier 的机会成本约等于放弃 1-2 年的研究推进速度。痛感明确。
