# 任务：为《狐狸谷》16 座建筑各重写一句 tip

上一轮 35 条里 19 条被采用，剩 16 条全被驳回，需要换角度重写。

## 写作风格（很严，比上一轮再严）

- 长度 **7-12 个汉字**（中间逗号算 1 字），最多 13 字，**绝不超过 13**
- **白描 + 画面 + 反差 / 悖论 / 小动作 / 小观察**
- 一个**具体**的瞬间——具体的物件、具体的声响、具体的气味、具体的动作
- 主语可以是：建筑本身 / 它（指狐狸）/ 物件 / 一种共存关系
- **禁词**：突然 / 忽然 / 能量 / 力量 / 神奇 / 无比 / 感慨 / 欣慰 / 惊喜 / 奇迹 / 沉默下来 / 安静下来 / 沉默了一会儿
- **禁套式**：`不是X，是Y` / `不是X而是Y` / `是X，不是Y` / `X，是X` 这种自我循环句
- **禁机制术语**：解锁 / 激活 / 系统 / 分支 / 阶段 / 版本 / 配方 / 产出 / 效率 / 上限 / 容量
- **禁抄 desc**：句子不能跟下面给出的描述近似复述
- 不要解释建筑功能，只**侧写**

## 角度建议（试不同侧切，每条都要换角度）

- **烟囱**：高度感、烟与天的距离、谷里抬头的视角、烟与风的拉扯
- **净化池**：水的耐心、池底的积淀、晨与夜的水面变化
- **钢仓**：堆叠的形态、温差、爪印、雨痕、月光
- **油缸**：密封感、缸壁的温度、它路过时的气味、油的"分量感"
- **蒸汽机房**：温度湿度、玻璃上的水汽、毛皮的反应、节拍感
- **内燃机**：震动传到旁物、惊扰的小动物、它的躲避动作
- **铁路**：枕木、远去的声音、轨上的小生物、行人留下的物
- **风力塔**：仰望、叶片的影子在地上扫过、风的方向、孤独感
- **精炼厂**：液态金属的流动、混合后的颜色变化、它分不清成分
- **钛库**：进出的温差、毛上的霜、库门的重量
- **熔炉**：火光的稳定性、它的眼睛适应、炉边的安宁
- **辉匣**：移动它、抱它、缝里漏出的光、与黑夜的对比
- **共振塔**：声波传到身体、远处物件的呼应、绕塔
- **辉映台**：丝走过的痕迹、留下的光痕、台面的纹路
- **灵核炉**：第一次搏动、等待的耐心、合上之后的活、温度
- **净辉林**：影子的消失、月光与辉芒并存、林中的风的不同

## 输出格式

16 行，每行：`id → tip`，比如：

```
chimney → ...
purifier → ...
...
```

只输出 16 行，**不要前言、不要解释、不要总结**。
