# 任务：为《狐狸谷》12 座建筑各重写一句 tip（**活泼版**）

上一轮 16 条里 4 条被采用，剩 12 条全部被嫌"太安静"，需要换成更**活泼/有动作/有声响/有调皮感**的写法。

## 关键反馈

> "其他都还不行，就不能写得活泼一些吗，这些都太安静了"

之前的版本都是静态白描——"水沉一夜""光自己走""影子站稳"……
现在要：**有动作、有声音、有小狐狸的反应、有拟人/夸张的小俏皮**。

## 写作风格

- 长度 **7-12 个汉字**，最多 13 字
- **要活泼**：动起来 / 响起来 / 跳起来 / 拟人化
- 主语优先用：**它（狐狸）做某个具体动作** / 或 **物件做出反应**
- 可以有：声响（叮、咚、扑、嗡、咕嘟）、小动作（跳、推、摇、探头、跑）、拟人（笑、躲、吵、抱、皱脸）、夸张（连凳子都坐不稳）
- **禁词**：突然 / 忽然 / 能量 / 力量 / 神奇 / 无比 / 沉默下来 / 安静下来
- **禁套式**：`不是X，是Y` / `X，是X`
- **禁机制术语**：解锁 / 激活 / 配方 / 产出 / 效率
- 不要解释建筑功能

## 用户已通过的"活泼"范例（钉死风格，重点学这些！）

```
内燃机   → 一动起来，连凳子都坐不稳。     (夸张 + 旁物反应)
共振塔   → 绕塔三圈，耳根开始发痒。       (动作 + 身体反应)
蒸汽机房 → 屋顶下雨，地上发烫。           (反差 + 画面)
祭坛     → 香在烧，它的影子也跪着。       (拟人)
矿坑     → 往下挖，光线短得只剩一截。     (具象动作 + 反差)
高炉     → 炉口的热风，吹乱了它的胡须。   (动作 + 身体)
工厂     → 一个动作做了一千遍。           (夸张)
邦交官   → 邦书很薄，压在上面的是两国的鼾声。 (荒诞俏皮)
使者     → 它的茶还没凉，客人已是旧友。   (时间反差)
契灵使   → 空房间里，它对着空气点头。     (滑稽)
精炼师   → 守着炉子，眼睛被火光映得发亮。 (动作 + 身体)
邦约     → 字没几个，夜倒长了。           (反差)
```

注意：**有狐狸的动作、有物件的反应、有声响、有夸张、有反差**——这些才"活"。

## 上一轮被驳回的"太安静版本"（**别学**，这种感觉就是"安静"）

```
[烟囱] 烟一出去，就忘了来路。 / 烟升到一半，被风撕成丝。 / 烟囱口的烟，够不着云。
[净化池] 一勺水添下去，黑就退一寸。 / 池底淤泥里，气泡悠悠浮起。 / 池底的泥，自己会沉。
[钢仓] 一推门，钢就响一声闷雷。 / 钢堆上，雨水聚成一颗颗圆珠。 / 月光铺满钢堆，没有一块是歪的。
[油缸] 拍一拍缸，里面咕嘟一下。 / 油味从缸缝渗出，黏住鼻尖。 / 铁壁摸上去，汗就干了。
[铁路] 它把耳朵贴在轨上，等远方。 / 枕木间，一株野花被风压低了头。 / 枕木缝里，长出几根不认识的草。
[风力塔] 叶片转着，云也排成一列。 / 叶片转一圈，地上的影子扫过它。 / 塔影转得比风慢。
[精炼厂] 出炉了，它分不清谁是谁。 / 铁水流入钛液，颜色慢慢变沉。 / 钢水和钛水，在炉里还分着层。
[钛库] 进库一趟，胡须上结了霜。 / 它从库中出来，胡须结了一层霜。 / 进去一层霜，出来一身汗。
[熔炉] 炉火不动，光从缝里慢慢淌。 / 炉火蓝蓝的，它盯了很久没眨眼。 / 火舌很稳，只偶尔舔一下炉口。
[辉匣] 抱着走，怀里就亮一截路。 / 匣子缝隙里，一丝光爬过它的指节。 / 夜里抱着它走，脚前有光。
[辉映台] 一根丝过台，留下一路亮。 / 光从丝上滑落，台面铺了一层亮。 / 台面的纹路，与晶丝走过的痕迹重合。
[净辉林] 林里没风，叶子自己在亮。 / 林中的光比月光明亮，影子很淡。 / 林子里的影子，到了午后就淡了。
```
