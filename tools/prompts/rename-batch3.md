# 任务：为《狐狸谷》研究重命名（第三批：灵修线 44 个）— 中式密教风（简洁版）

灵修线要的是**中式密教模拟器**的气质。上一轮跑偏了——往复杂辞藻堆砌走（"灵犀乍动""伏脉寻踪""同频玄应""振彻魂魄""元念镇岳"），那不是密教，那是堆砌。

## 核心：密教的魅力是「简」

> 简洁，又有神秘感，不用复杂词汇，富有灵气 —— 才是密教的魅力。

参照《密教模拟器》(Cultist Simulator)：它的命名是 **Lantern（灯）、Moth（蛾）、Knock（叩）、Forge（炉）、Grail（杯）、Winter（冬）** —— 全是**最简单的词**，靠**留白、语境、暗示**生出神秘感和灵气。神秘不靠生僻字堆出来，靠"简 + 余味"。

### 反面示范（上一轮的错误，绝对别学）
```
灵犀乍动 伏脉寻踪 澄心止念 同频玄应 玄谱析真 振彻魂魄
元念镇岳 心镜漏字 灵觉破障 念动无源 暗契随身 三镜照非
```
—— 太满、太用力、堆砌"玄/谧/骸/彻/镇/破"这种生僻字，没有灵气，只有沉重。

### 正面方向（简洁有灵气）
灵修线**现名本身就有不少是对的**——「净心」「铭刻」「灵酿」「辉映」「念阵」「净辉」「星感」「超感」「灵网」「幽理」「元驱」「镜锻」「灵驱」「净寂」——这些 2-3 字、简洁、有灵气、有神秘感，**该保留就保留**。

## 命名规则

1. **简！2-3 个字为主**（4 字只在确实精炼时用），不堆砌
2. **只用常见、轻盈、有灵性的字** —— 不用"玄/谧/骸/彻/镇/破/枢"这类生僻或沉重的字
3. **神秘感靠留白和余味**，不靠辞藻
4. **现名已经简洁有灵气的 → 直接保留**（该行写现名即可）
5. 真正要改的是**"教科书味"**的——"感应术""灵脉学""共鸣术""化形基础""悟道法""通灵术""形魄学""寂界术""灵契学""元念术""结晶术"这种 "X术/X学/X法/X基础" —— 去掉教科书后缀，或换一个更轻灵的说法
6. 不要工业科技词（那是工业线）
7. 阶段由浅入深：初感 / 共鸣 / 化形 / 深寂
