# 任务：为《狐狸谷》35 座建筑各写一句 tip

《狐狸谷》是一个静谧氛围的中式叙事增量游戏，主角是住在山谷里的小狐狸。
建筑都坐落在它的谷里，由它管理、由它使用。

## 写作风格（很严格）

- 长度 **7-13 个汉字**（中间逗号算 1 字），上下浮动 ±1 也行，但绝不超过 14 字
- **白描 + 画面 + 反差 / 悖论 / 小动作 / 小观察**
- 一个**具体**的瞬间——具体的物件、具体的声响、具体的气味、具体的动作
- 主语可以是：建筑本身 / 它（指狐狸）/ 物件 / 一种共存关系
- **禁词**：突然 / 忽然 / 能量 / 力量 / 神奇 / 无比 / 感慨 / 欣慰 / 惊喜 / 奇迹 / 沉默下来 / 安静下来 / 沉默了一会儿
- **禁套式**：`不是X，是Y` / `不是X而是Y` / `是X，不是Y` / `X，是X` 这种自我循环句
- **禁机制术语**：解锁 / 激活 / 系统 / 分支 / 阶段 / 版本 / 配方 / 产出 / 效率 / 上限 / 容量
- 不要解释建筑功能，只**侧写**——读者已经知道它做什么了
- 不要"狐狸"二字反复出现；用"它"即可，或者干脆不出现主语
