# 任务：为《狐狸谷》增量编写 23 条山谷见闻事件（每条 3 条文风候选）

你是资深独立游戏文案作者，擅长长句白描、密教氛围、悖论式叙事。

为放置类增量游戏《狐狸谷》（一群狐狸在山谷里建文明）写 23 条「山谷见闻」事件。**每条事件出 3 条文风候选**（A/B/C），以便后期挑选。

---

## 一、文风硬约束

### 三种参考文风（每条事件分别出一条）

**A 类 · 类极乐迪斯科（Disco Elysium）**
- 内心独白渗透到外景描写里
- 句式带哲学感、自反、黑色幽默
- 长句 + 逗号串联多个分句
- 物件被拟人化，但带讽刺距离感
- 例（仿写）：「一只老斥候看着自己布满旧伤的脚掌，认真考虑了一会要不要给每一道伤起个名字，最后决定算了——它们彼此之间已经形成了一种默契的沉默。」

**B 类 · 类密教模拟器（Cultist Simulator）**
- 神秘氛围 + 仪式动作 + 不解释的象征
- 长句堆叠隐喻，避免说穿
- 物件具备半神性（坛子在等待，火焰在记得）
- 时间感模糊（"很久以前的某个下午" / "在它发生之前就已经发生过的"）
- 例（仿写）：「圣火窑的炉膛今夜没有熄灭，火焰在木炭已经燃尽很久之后仍以同一个角度向左偏斜，像在听某个尚未到达的声音；圣工匠没有添柴，他只是站在边上等。」

**C 类 · 类博尔赫斯（Borges）**
- 镜面、迷宫、地图、无限、记忆——抽象意象具象化
- 长句嵌套，从句叠从句
- 时间循环、身份倒影、文本即世界
- 学者气质 + 神话感
- 例（仿写）：「学者发现卷轴的第三十七页所记载的内容，与他三个月前抄写却已遗失的另一卷其中一段，逐字相同；他想不起是抄写时记错了页码，还是更早某个未曾经历的日子里，已经写过这一句。」

### 共同硬约束（所有候选都必须满足）

1. **长句**：每条事件 **40-100 字**，多分句嵌套，**禁止短促句**（如"风停了。月亮亮。"这种节奏一律不要）
2. **角色具体**：永远是某只具体的狐狸——采集者、伐木工、矿工、祭司、感知者、铁匠、商贩、斥候、长者、幼狐、化形师、钻井工、精炼师、织丝师、圣工匠、狂信者 等。**禁用"狐狸们"/"村民"/"人们"**（少用复数大群体）
3. **代词硬规则**：所有狐狸**只用"它"**，**禁止使用"他""她""他们""她们"**。即使描写有性别暗示的角色（祭司/长者/幼狐 等），仍统一用"它"。这是物种级硬约束。
4. **白描底色**：再玄也要从一个物件或一只狐狸的身体动作起笔
5. **避免词汇**：能量、力量、神奇、神秘、奇迹、感慨、欣慰、惊喜、突然、忽然、无比、神奇地
6. **不要总结尾巴**：不写"这是好兆头"、"狐狸们感到很高兴"这种闭合句
7. **物件具体**：年轮、石壁、卷轴、爪印、坛子、签文、扁圆孔、卡槽、釉质、藤蔓——避免"美丽的景象"

## 二、要写的 23 条（按 5 个分组）

### A. 教团事件（5 条，sb: 'D'，br: 'I'，"工业 + 神启"玩家）

教团 = 中世纪教廷式组织化宗教，狂热信徒、圣工匠、教令系统、圣火/圣铁。建筑：圣火窑(holyKiln)、圣工坊(holyForge)、教令堂(edictHall)、审判庭(tribunalHall)、圣铁库(holyIronVault)、圣油坊(oilPress)。

5 条主题：
1. 教令颁布／执行（新教令贴出引发狐狸围观）
2. 圣火试炼／圣火窑异象
3. 信仰审判／审判庭场面
4. 狂信者集会／集体仪式
5. 圣工匠锻造圣铁的瞬间

### B. 秘仪事件（5 条，sb: 'D'，br: 'M'，"灵修 + 神启"玩家）

秘仪 = 个人化密教修行走向飞升，神露/秘知、化神池(apotheosisPool)、禁典阁、秘仪殿(mysteryHall)、圣林(sacredGrove)。氛围安静、玄、个人内省。

5 条主题：
1. 秘仪殿独修者出关时刻
2. 神露异象／化神池微光
3. 秘知碎片浮现（一只狐狸读懂了不该懂的字）
4. 圣林的某一棵树
5. 飞升前夕的"门前"瞬间（飞升阶梯第一/二门）

### C. 邦交事件（3 条，sb: 'T'）

邦交 7 族：山猫(wildcat)、河獭(otter)、白鹤(crane)、旧墟遗民(ruinfolk)、雪鸮(owl)、水鼠(ratter)、考拉(koala)。每族浅盟（深度 1-2）已实装。

3 条主题：
1. 邦书签订／盟邦正式建立
2. 异珍献礼（某族送来的稀奇物件）
3. 远客到访（某族驻谷代表的日常瞬间）

### D. 工业 C 阶段事件（5 条，br: 'I'，phase: 4）

工业 C = 精金时代。寒钛、合金、纲要、钛构件、混凝柱、星图。建筑：煅烧炉(calcFurnace)、精炼厂(refinery)、望远台(observatory)、净林(cleanForest)、钛库(titanVault)。职业：精炼师(refiner)。

5 条主题：
1. 煅烧炉的高温／精炼师的瞬间
2. 寒钛与合金的诞生
3. 望远台第一次看到的星象
4. 净林反治污染的微小迹象
5. 钛库 / 混凝柱建筑的物理感

### E. 灵修 C 阶段事件（5 条，br: 'M'，phase: 4）

灵修 C = 化形时代。晶丝、辉芒、灵核、形魄、灵图。建筑：结晶窟(crystalCave)、辉映台(radianceDais)、灵核炉(coreForge)、净辉林(radiantGrove)、灵图阁(chartHall)。职业：化形师(shapeMaster)。

5 条主题：
1. 化形师的形魄变化
2. 灵核诞生的瞬间
3. 辉芒映照下的山谷
4. 灵图阁里浮现的图案
5. 结晶窟深处的晶丝活动

## 三、输出格式（严格遵守）

### 整体结构

```
# Phase 4.6 事件 23 条 × 3 文风候选

## A. 教团事件（5 条）

### A1. 教令颁布／执行
- **A 类 · 极乐迪斯科风**：
  { t: 'XXX（40-100字长句）', sb: 'D', br: 'I', uq: { b: { edictHall: 1 } }, w: 2 },
- **B 类 · 密教模拟器风**：
  { t: 'XXX', sb: 'D', br: 'I', uq: { b: { edictHall: 1 } }, w: 2 },
- **C 类 · 博尔赫斯风**：
  { t: 'XXX', sb: 'D', br: 'I', uq: { b: { edictHall: 1 } }, w: 2 },

### A2. 圣火试炼／圣火窑异象
（同上 3 候选）
...
```

### 字段说明
- `t`: 事件文本（长句 40-100 字，多分句）
- `sb` / `br` / `phase`: 分支门控（按上方分组固定）
- `uq`: 解锁条件（按上方建议建筑/职业 ID）
- `e`: 资源效果（可选；多数为 1-3 单位，少数事件可不给）
- `w`: 权重（1=罕见，2=常见，3=很常见；建议 1-2 为主）

### 资源 ID 速查表（写 e 时只能用这些）
- 基础: berry, wood, stone, leather, iron
- 加工/工业: coal, steel, plate, concrete, gear, oil, barrel, titan, alloy, outline, titanPart, pillar, starchart
- 知识: scroll, lore, draft
- 贸易: coin, charm, remnant, spice, silk, ancCoin, dye, wine, ink
- 灵修: spirit, fateSilk, bead, spiritInk, sigil, resonance, elixir, spectrum, insight, crystalSilk, radiance, spiritCore, formSoul, spiritChart
- 神启: piety, holyOil, holyFlame, holyIron, ambrosia, gnosis
- 通达: renown, credential, charter, exotic
